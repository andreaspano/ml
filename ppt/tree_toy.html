Basic

We have a sample of ten events:

event <-  c(1, 1, 1, 1, 0, 1, 1, 0, 0, 0)
event 
##  [1] 1 1 1 1 0 1 1 0 0 0

We wish to classify the whole sample as either zero or one.

As the majority of the set is made by events of type one, we classify the whole sample as a sample of class one.

The Miss Classification Error (MCE) being:

4/10
## [1] 0.4

as four observations, four zeros, are wrongly classified

toy tree

fm <- rpart(event~gender, 
    data = data, 
    method = 'class',
    parms = list(split = 'gini'), 
    control = rpart.control(
        minbucket = 2,
        cp = 0.00001),
    xval = 2)
rpart.plot(fm)
